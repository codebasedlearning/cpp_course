# Exam - Prep

---

In der Prüfung wird in der Regel (!) genau eine Datei bearbeitet. Das ist auf
dem Aufgabenblatt beschrieben – bitte unbedingt ganz genau lesen!

Der Dateiname lautet `a1234_Matrnr_Name.cpp` oder so ähnlich, wobei Sie hier
den Namen und die Matrikelnummer ersetzen müssen. Erledigen Sie das, bevor
Sie beginnen, damit Sie direkt mit der richtigen Datei arbeiten!

Ob Sie ein Makefile benötigen oder den Compiler direkt aus dem Editor oder
Terminal starten können, hängt an der Konfiguration des Rechners und in
welcher Umgebung Sie die Prüfung ablegen.

Die Datei `a1234_Matrnr_Name.cpp` können Sie als Vorlage auf einem Stick 
mitbringen. Da sind zumindest eine Menge `include`s enthalten und eine grobe
Struktur mit Aufgaben (`A1` etc.) und `main`. Dort müssen Sie auch unbedingt
Ihren Namen und die Matrikelnummer im Kopf angeben!
